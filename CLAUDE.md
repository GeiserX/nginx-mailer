# CLAUDE.md — nginx-mailer

## Overview
Lightweight Docker image based on nginx:alpine that serves static websites and handles contact form submissions via a built-in Go API with SMTP email support and optional Cloudflare Turnstile CAPTCHA verification.

## Tech Stack
- Go (API backend)
- nginx:alpine (static file serving)
- supervisord (process management)
- Docker (multi-arch: linux/amd64, linux/arm64)
- GitHub Actions CI

## Development
```bash
# Build the Go API
cd cmd/mailer && go build -o /tmp/mailer .

# Run locally with Docker
docker compose -f docker-compose.example.yml up

# Tests
go test ./...
```

## Architecture
- `cmd/mailer/` — Go API entrypoint
- `internal/email/` — SMTP email sending logic
- `internal/handler/` — HTTP request handlers for the contact form API
- `internal/turnstile/` — Cloudflare Turnstile CAPTCHA verification
- `nginx/` — nginx configuration files
- `supervisord.conf` — Process manager for nginx + Go API in single container
- `docs/` — Documentation and images

## Key Rules
- Never hardcode SMTP credentials; pass via environment variables
- Docker image published to Docker Hub as `drumsergio/nginx-mailer` with semver tags
- Single container runs both nginx and the Go API via supervisord
- Cloudflare Turnstile is optional (enabled only when `CLOUDFLARE_TURNSTILE_SECRET_KEY` is set)
